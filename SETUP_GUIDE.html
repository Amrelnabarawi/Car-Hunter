1

إنشاء Repository على GitHub

1

افتح GitHub وسجل دخول

روح على github.com وسجل دخول (أو أنشئ حساب مجاني لو مفيش)

2

أنشئ Repository جديد

اضغط New repository وسميه car-hunter واختار Private ثم اضغط Create repository

3

ارفع ملفات المشروع

اضغط uploading an existing file وارفع الملفات الثلاث:

📁 car-hunter/ ├── car_hunter.py ├── requirements.txt └── .github/ └── workflows/ └── car_hunter.yml
⚠️
مهم! ملف car_hunter.yml لازم يكون في مجلد .github/workflows/ — GitHub لازم تنشئ المجلدات دي يدوياً عند الرفع
2

إعداد Telegram Bot

1

ابحث عن BotFather في Telegram

افتح Telegram وابحث عن @BotFather وابدأ محادثة معاه

2

أنشئ Bot جديد

ابعت الأمر ده:

/newbot

هيطلب منك اسم للبوت، اكتب مثلاً: Car Hunter Amr

بعد كده هيطلب username ينتهي بـ bot مثلاً: car_hunter_amr_bot

3

احفظ الـ Token

BotFather هيبعتلك رسالة فيها Token يبدو كده:

1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

احفظ الـ Token ده — هتحتاجه في الخطوة 4

4

احصل على Chat ID بتاعك

افتح البوت الجديد وابعت له أي رسالة، بعدين افتح الرابط ده في المتصفح (عدل الـ TOKEN):

https://api.telegram.org/botYOUR_TOKEN/getUpdates

دور على "chat":{"id": — الرقم اللي بعدها هو الـ Chat ID بتاعك

3

إعداد Gmail App Password

ℹ️
محتاج تنشئ App Password خاص — مش كلمة المرور الأصلية بتاعتك على Gmail
1

فعّل التحقق بخطوتين

روح على myaccount.google.com/security وفعّل 2-Step Verification

2

أنشئ App Password

في نفس الصفحة، دور على App passwords وضغط عليها

في خانة الاسم اكتب: Car Hunter ثم اضغط Create

3

احفظ الـ Password

هيظهرلك password من 16 حرف كده:

xxxx xxxx xxxx xxxx

احفظه — هتحتاجه في الخطوة الجاية

4

إضافة GitHub Secrets

في الـ Repository بتاعك على GitHub: اضغط Settings → Secrets and variables → Actions → New repository secret

اسم الـ Secret القيمة مصدرها
TELEGRAM_BOT_TOKEN Token من BotFather الخطوة 2
TELEGRAM_CHAT_ID رقم Chat ID بتاعك الخطوة 2
GMAIL_USER إيميلك الكامل Gmail بتاعك
GMAIL_APP_PASSWORD الـ 16-character password الخطوة 3
NOTIFY_EMAIL الإيميل اللي هتستقبل فيه ممكن نفس Gmail
ANTHROPIC_API_KEY API Key من Anthropic console.anthropic.com
⚠️
أكتب اسم الـ Secret بالضبط زي ما مكتوب في الجدول — بالأحرف الكبيرة والـ underscore
5

تشغيل البوت للمرة الأولى

1

افتح Actions في GitHub

في الـ Repository، اضغط على Actions من القائمة العلوية

2

شغّل الـ Workflow يدوياً

اضغط على 🚗 Car Hunter — Hourly Search ثم اضغط Run workflow → Run workflow

3

تابع النتيجة

انتظر 2-3 دقيقة ثم راجع Telegram والإيميل — لو وجد سيارات هتوصلك إشعار فوري!

4
